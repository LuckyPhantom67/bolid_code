## Про команду

Ми — команда **Lucky Phantom**, яка створила власний болід для проходження трас на швидкість та точність.  
Проєкт поєднує програмування, електроніку, PID-регулювання та оптимізацію руху в реальному часі.

## Ідея проєкту

Основна задача боліда — максимально швидко та стабільно проходити трасу, використовуючи датчики лінії та алгоритм PID-керування.

Болід аналізує положення чорної лінії відносно центру датчиків і в реальному часі змінює швидкість моторів для точного проходження поворотів та прямих ділянок.

Проєкт створювався з акцентом на:

- стабільність руху
- високу швидкість
- плавне проходження поворотів
- автоматичне калібрування датчиків
- підтримку реверсу моторів
- динамічне прискорення на прямих ділянках

---

# Конструкція боліда

У проєкті використовуються:

- Arduino Nano
- QTR line sensors
- драйвер моторів
- два DC-мотори
- акумуляторне живлення
- система PID-керування

Болід був налаштований спеціально під трасу з різкими та плавними поворотами.

---

# Програмна частина

Код побудований на основі:

- Finite State Machine
- PID Controller
- QTRSensors library

Основні режими роботи:

1. Очікування калібрування
2. Автоматичне калібрування
3. Очікування старту
4. Рух по лінії
5. Автоматична або ручна зупинка
